# Isaac Neibaur - Portfolio Source (Dev)

This repository contains the source code for [isaacneibaur.com](https://isaacneibaur.com). 
It uses the **Hugo Blox (Academic)** theme.

## 🛠 Tech Stack
- **Framework:** Hugo (Extended Version)
- **Styling:** Tailwind CSS
- **Automation:** GitHub Actions
- **Hosting:** GitHub Pages (via cross-repo deploy)

## 🚀 How to Make Changes
1. **Open VS Code** in this directory.
2. **Start Local Server:** Run `hugo server` in the terminal.
3. **Preview:** View changes live at `http://localhost:1313`.
4. **Deploy:** Commit and push to the `main` branch. 
   - *Note: Pushing to `main` triggers a GitHub Action that builds the site and pushes it to the production repo.*

## 📁 Key Folders
- `assets/media/authors/me.png`: Your profile picture.
- `content/`: Main content folder for all pages.
  - `content/_index.md`: Homepage content and configuration.
  - `content/publications/`: Individual publication pages.
  - `content/events/`: Individual talk/event pages.
  - `content/project/`: Individual portfolio project pages.
- `static/uploads/`: Store your Resume PDF here.
